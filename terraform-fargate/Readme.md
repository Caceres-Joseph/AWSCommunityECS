# Terraform Fargate Setup

Este proyecto utiliza **Terraform v1.13.3** para desplegar infraestructura en AWS Fargate.

## Requisitos

- [Terraform v1.13.3](https://www.terraform.io/downloads.html)
- Credenciales de AWS configuradas

## Pasos para desplegar

1. Inicializa el proyecto:
    ```bash
    terraform init
    ```

2. Revisa el plan de despliegue:
    ```bash
    terraform plan
    ```

3. Aplica los cambios para desplegar:
    ```bash
    terraform apply
    ```

## Notas

- Asegúrate de tener permisos suficientes en tu cuenta de AWS.
- Consulta la documentación oficial de Terraform para más detalles.
