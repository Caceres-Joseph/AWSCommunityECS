# AWSCongreso2025


## Cómo empezar

1. Clona el repositorio:
    ```bash
    git clone ...
    ```
2. Instala las dependencias necesarias según la documentación del proyecto.
3. Realiza tus cambios y súbelos al repositorio:
    ```bash
    git push -u origin main
    ```
